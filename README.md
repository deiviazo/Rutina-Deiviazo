# Rutina Deiviazo PWA

Esta es una app nueva e independiente basada en la estetica de la rutina anterior, con cuatro dias:

- Pecho + triceps
- Hombro + biceps
- Pierna + abdomen
- Espalda

El historial se guarda en el propio dispositivo mediante `localStorage`, con claves independientes: `deiviazo-v1-storage` y `deiviazo-v1-history`.

## Subir a Vercel

Sube el contenido de esta carpeta a un repositorio de GitHub e importalo en Vercel como proyecto `Other`.

- Build Command: vacio
- Output Directory: vacio o `.`
